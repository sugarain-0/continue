# 📱 Spacedesk 教程：将手机/旧电脑变成副屏

> **屏幕扩展与投屏 (Windows PC to Mobile / PC to PC)**
> 
> 这是一个关于如何使用 **spacedesk** 软件，将移动设备（安卓/iOS）或其他电脑（Windows/Mac/Linux）变为 Windows 电脑副屏的详细教程。

---

## 🛠 准备工作

在开始之前，请确保你拥有以下设备：
* **主控端 (Server/Primary Machine):** 一台安装了 Windows 10/11 的电脑（信号源）。
* **接收端 (Viewer/Secondary Machine):** * 手机/平板 (Android / iOS)
    * **或者** 另一台电脑 (Windows / Mac / Chromebook)
* **连接环境:** * **无线连接：** 两台设备需连接在同一个 Wi-Fi 网络下。
    * **有线连接：** 一条可靠的 USB 数据线（仅限手机连接时）。

---

## 📥 第一步：安装主电脑驱动 (Server)

主电脑作为“信号发射源”，必须安装驱动程序。

1.  访问 [spacedesk 官方网站](https://www.spacedesk.net/)。
2.  下载 **spacedesk Driver Software for Windows**。
3.  **注意：** 必须安装在系统盘（C盘），**不支持直接安装到 U盘 (USB drive)** 运行，因为它包含系统级驱动。
4.  安装时勾选“添加防火墙例外”，安装完成后**重启电脑**。
5.  确保右下角系统托盘图标显示为 **ON**。

---

## 💻 第二步：设置接收端 (Viewer)

根据你想用的接收设备选择对应的操作：

### 1. 如果接收端是 手机/平板
* 在应用商店搜索并安装 `spacedesk`。

### 2. 如果接收端是 另一台电脑 (电脑投屏电脑)
这是 Spacedesk 最强大的地方，你有两种选择：
* **客户端模式 (Windows 专用)：** 在官网下载并安装 `spacedesk Viewer for Windows`。
* **网页版模式 (跨平台通用 - Mac/Linux/Win)：** 不需要安装任何软件，直接在副电脑的浏览器中访问：[**https://viewer.spacedesk.net/**](https://viewer.spacedesk.net/)。

---

## 🔗 第三步：连接与投屏

### 1. Wi-Fi 连接（最方便）
1.  确保两台设备在**同一个 Wi-Fi** 下。
2.  打开接收端（手机 App 或电脑 Viewer / 网页版）。
3.  当看到 `Connection: [主电脑IP地址]` 时，点击它。
4.  **电脑投屏电脑** 成功，副电脑现在就是主电脑的第二个显示器了！

### 2. USB 连接（手机专用 - 零延迟）
1.  使用数据线连接手机和电脑。
2.  在手机设置中开启 **USB 网络共享 (USB Tethering)**。
3.  在手机 App 中点击识别出的 IP 即可连接。

---

## ⚙️ 进阶设置 (优化使用体验)

1.  **排列位置：** 在主电脑桌面右键 -> **显示设置**，拖动屏幕图标对齐物理位置。
2.  **扩展模式：** 在“多显示器设置”中选择 **“扩展这些显示器”** (Extend)。
3.  **性能优化：** 如果副电脑或手机画面卡顿：
    * 在 Viewer 端（手机或副电脑软件内）找到 `Settings`。
    * 降低 **Color Depth** (色深) 或 **Image Quality** (画质)。
    * 将帧率设置为 30 FPS。

---

## ❓ 常见问题排查 (FAQ)

* **Q: 为什么不能装在 U盘里运行？**
    * A: spacedesk 需要向 Windows 系统内核注入虚拟显示驱动，U盘无法提供系统级别的服务运行环境。
* **Q: 电脑投屏电脑搜不到 IP？**
    * A: 请检查主电脑的防火墙是否开启。如果是网页版投屏，请确保浏览器允许弹出窗口。
* **Q: 画面有水印吗？**
    * A: 目前个人非商业用途是免费的，没有功能限制。
