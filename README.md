# Scrcpy 手机投屏工具

## 下载
- 点击进入 'scrcpy-win64-v3.3.4' 文件夹下载所有文件 

## 使用方法
1. 解压下载的文件  #裝在 usb/電腦硬碟 也可以。
2. 手机开启 USB 调试（设置→关于手机→连续点击版本号7次→返回开发者选项→开启USB调试）
3. 进入解压后的文件夹，**双击运行 'scrcpy.exe'**

## 熄屏模式（手机屏幕关闭，电脑继续操作）

### 方法一：使用快捷方式（推荐，一劳永逸）
1. 在 `scrcpy-win64-v3.3.4` 文件夹里，右键 → 新建 → 文本文档
2. 重命名为 `启动熄屏.bat`（注意扩展名是 `.bat`）
3. 右键编辑这个文件，粘贴下面这行代码：
   ```bat
   scrcpy --turn-screen-off
   pause
   ```
4. 保存后，以后**双击这个 `.bat` 文件**即可启动

### 方法二：每次手动输入命令
1. 进入 'scrcpy-win64-v3.3.4' 文件夹
2. 在文件夹的**地址栏**输入 'cmd' 回车
3. 在黑窗口里输入：
   #cmd
   scrcpy --turn-screen-off


##常见问题
提示找不到设备	① 检查数据线是否插好 ② 手机屏幕要解锁 ③ 重新插拔USB
手机上弹窗询问	勾选「一律允许」，点击确定
鼠标键盘无法控制	开发者选项里打开「USB调试（安全设置）」（OPPO/小米/vivo需要）
