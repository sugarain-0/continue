# Scrcpy 手机投屏到電腦工具

## 下载
- 点击进入 `scrcpy-win64-v3.3.4` 文件夹下载所有文件

## 使用方法
1. 解压下载的文件（裝在 usb/電腦硬碟 也可以）
2. 2. 手机开启 USB 调试（设置→关于手机→连续点击版本号7次→返回开发者选项→开启USB调试）
3. 进入解压后的文件夹，**双击运行 `scrcpy.exe`**

## 熄屏模式（手机屏幕关闭，电脑继续操作）

### 方法一：使用快捷方式（推荐，一劳永逸）
1. 在 `scrcpy-win64-v3.3.4` 文件夹里，右键 → 新建 → 文本文档
2. 重命名为 `启动熄屏.bat`（注意扩展名是 `.bat`）
3. 右键编辑这个文件，粘贴下面这行代码：
   ```bat
   scrcpy --turn-screen-off
   pause
   ```
4. 保存后，以后**双击这个 `.bat` 文件**即可启动

### 方法二：每次手动输入命令
1. 进入 `scrcpy-win64-v3.3.4` 文件夹
2. 在文件夹的**地址栏**输入 `cmd` 回车
3. 在黑窗口里输入：
   ```cmd
   scrcpy --turn-screen-off
   ```
常见问题
1. 出现「ERROR: Device is unauthorized」
原因：手机未授权电脑
解决：

看手机屏幕，弹出「允许USB调试」窗口时，勾选「一律允许」，点击确定

如果没有弹窗，在开发者选项里关闭再重新打开「USB调试」

2. 出现「ERROR: Could not find any ADB device」
原因：电脑没检测到手机
解决：

确保手机屏幕已解锁

重新插拔USB数据线

换个USB接口（台式机插机箱背面）

换一条能传数据的数据线（不是只能充电的线）

重启手机

3. 鼠标/键盘无法控制手机
原因：部分手机需要额外开启安全设置
解决：

进入「开发者选项」→ 找到「USB调试（安全设置）」→ 开启

然后重启手机

（常见于OPPO、小米、vivo手机）

4. 画面卡顿或延迟高
解决：

降低分辨率：用命令 scrcpy --max-size 1024

降低码率：用命令 scrcpy --bit-rate 2M

关闭手机其他后台应用

5. 连接后手机自己断开
解决：

换一根USB数据线

换个USB接口

检查手机USB接口是否松动

6. 双击 .bat 文件闪退
原因：没有在正确目录运行
解决：

确保 .bat 文件放在 scrcpy-win64-v3.3.4 文件夹里面

或者右键编辑 .bat，改成：

bat
cd /d D:\你的路径\scrcpy-win64-v3.3.4
scrcpy --turn-screen-off
pause
退出程序
直接关闭 Scrcpy 窗口

或在黑窗口里按 Ctrl + C
